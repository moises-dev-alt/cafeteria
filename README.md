# Cafeteria React (simulado)

Projeto React minimal para uma cafeteria com carrinho e checkout simulado.

Como rodar

1. Instale dependências:

```bash
npm install
```

2. Rode em modo desenvolvimento:

```bash
npm run dev
```

Funcionalidades

- Navegação simples (Home, Cardápio)
- Adicionar itens ao carrinho, ajustar quantidade
- Checkout simulado; pedidos salvos em `localStorage`

Observações

- Este projeto é uma base: para pagamentos reais, integre Stripe/PayPal e um backend seguro.
